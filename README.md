# Ebooks

Generate your own horse_ebooks by @busterbenson and compiled as a gem by @parkr

## Installation

Add this line to your application's Gemfile:

    gem 'ebooks'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ebooks
